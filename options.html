Robot Vision Control (RVC)

Thank you for using RVC! To get the most out of this extension, download the the software required for controlling the host machine: rvc.jar

  1. Launch the app, by double clicking the jar file.
  2. Click on Generate, to generate a new key.
  3. Configure more, if you want
  4. Click Run.

The RVC will be running in the background. You can close the window, and use the system tray icon to re-open it (if your OS supports this behavior).

Visit the application's github landing page for the intimate details: rvc
